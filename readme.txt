Implementação de um algoritmo de descrição de imagens, com o intuito de proporcionar uma experiência imersiva para turista com deficiência visual.


Foi escolhida uma arquitetura de codificador-decodificador, combinada juntamente com um mecanismo de atenção, para a construção do algoritmo de descrição de imagens. O codificador desempenha a função de extrair representações das imagens, enquanto o decodificador é encarregado de gerar legendas correspondentes.


Gerador-de-legenda.devImages.txt = Fragmento do conjunto de dados utilizado para Validação.
Gerador-de-legenda.testImages.txt = Fragmento do conjunto de dados utilizado para Teste.
Gerador-de-legenda.token.txt = Todas as legendas atreladas às imagens do dataset utilizado no projeto.
Gerador-de-legenda.trainImages.txt = Fragmento do conjunto de dados utilizado para Treino.
Gerador-de-legenda_text.zip = Arquivo .zip dos arquivos txt utilizados no projeto.
gerador-de-legendas.ipynb = Notebook do projeto utilizado.
requirements.txt = Requisitos de bibliotecas para o funcionamento do algoritmo.
